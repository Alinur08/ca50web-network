# CS50 Web – Network Project

## 📌 Description

This project is a social network web application built as part of **CS50’s Web Programming with Python and JavaScript** course.
Users can create posts, like posts, follow other users, and view different feeds.

---

## 🚀 Features

* Create new posts
* View all posts (global feed)
* Like and unlike posts
* Edit own posts
* Follow / unfollow users
* View user profiles
* See following feed (posts from followed users)
* Pagination for posts

---

## 🛠️ Technologies Used

* Python (Django)
* JavaScript (Vanilla JS)
* HTML & CSS
* SQLite

---

## ⚙️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Alinur08/cs50web-network.git
   cd cs50web-network
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run migrations:

   ```bash
   python manage.py migrate
   ```

4. Start server:

   ```bash
   python manage.py runserver
   ```

5. Open in browser:

   ```
   http://127.0.0.1:8000/
   ```
